# Explore a variety of astronomical concepts

Static HTML/CSS/JS pages for interactive Earth/planet visualizations (English and Arabic).

## Pages
- `index.html` / `earth-ar.html` (Earth)
- `earth-seasons.html` / `earth-seasons-ar.html`
- `earth-analemma.html` / `earth-analemma-ar.html`
- `planets.html` / `planets-ar.html`
- `venus.html` / `venus-ar.html`

## Local use
Open the HTML files directly in a browser, or serve this folder with any static server.

## Deploy
Configured for Netlify. See `_redirects`, `robots.txt`, and `sitemap.xml`.
